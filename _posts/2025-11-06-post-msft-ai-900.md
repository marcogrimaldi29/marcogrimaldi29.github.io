### üìä Skills Measured (Updated on May 2, 2025)

This section covers the **[skills measured](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-900#skills-measured-as-of-may-2-2025) for the AI-900 certification**.  
The domains are divided into five main knowledge areas, each weighted differently in the exam:

- **Describe Artificial Intelligence workloads and considerations (15‚Äì20%)**
  - Identify features of common AI workloads (e.g., prediction, classification, conversation, vision, language)
  - Describe guiding principles for responsible AI
  - Describe considerations for fairness, reliability, transparency, privacy, and safety in AI solutions

- **Describe fundamental principles of machine learning on Azure (15‚Äì20%)**
  - Identify core machine learning concepts (regression, classification, clustering)
  - Describe supervised, unsupervised, and reinforcement learning
  - Describe features and uses of Azure Machine Learning

- **Describe features of computer vision workloads on Azure (15‚Äì20%)**
  - Identify common computer vision tasks (image classification, object detection, facial recognition, OCR)
  - Describe Azure services for computer vision (Azure AI Vision, Custom Vision, Face, Form Recognizer)
  - Describe use cases for computer vision in Azure

- **Describe features of Natural Language Processing (NLP) workloads on Azure (15‚Äì20%)**
  - Identify common NLP tasks (sentiment analysis, language detection, translation, entity recognition)
  - Describe Azure services for NLP (Azure AI Language, Translator, QnA Maker)
  - Describe use cases for NLP in Azure

- **Describe features of generative AI workloads on Azure (20‚Äì25%)**
  - Identify generative AI concepts (large language models, image generation, conversational AI)
  - Describe Azure services for generative AI (Azure OpenAI Service, Copilot Studio)
  - Describe use cases for generative AI in Azure
